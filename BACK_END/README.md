## 使うための前準備
#### ライブラリのインストール
1. コマンドでフォルダ内に移動し
```bash
    $pip install -r requirements.txt
    または
    $py -m install -r requirements.txt
```
を実行する。

## デモ
ローカル環境での実行方法

```bash
    export FLASK_APP=application.py
    flask run
```
#### WindowsのCMDプロンプトの場合
```bash
    set FLASK_APP=application.py
    flask run
```
#### Windowsのパワーシェルの場合
```bash
    $env:FLASK_APP = "application.py"
    flask run
```

127.0.0.1:5000にアクセスすることでサーバを実行することができる.
